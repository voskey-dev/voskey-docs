# 投稿時の注意点

- **センシティブな内容の文章**を投稿する際は、公開範囲を「パブリック（🌐）」に設定したまま直接公開しないでください。
  - 必ず「内容を隠す（CW）機能👁️」などを利用し、注釈でセンシティブな内容であることを明記するなど、閲覧者が事前に内容を把握できるように配慮した上で公開してください。
- **センシティブな内容の画像や動画**を投稿する際は、事前に「センシティブとして設定👁️」機能を使用し、センシティブフラグを設定してから投稿してください。

<Aside type="note" title="「センシティブな内容」とは？">
- 「閲覧を希望しない方がいる可能性のある情報」を指します。
  - NSFW（Not Safe For Work：職場非安全）と表現されることもあり、職場や学校、公共交通機関など、公共の場での閲覧に適さない内容を意味します。
  - 一般的には性的な内容を指すことが多いですが、暴力的な表現、政治や宗教に関する話題、激しい光の点滅を伴う映像なども含まれます。
  - 詳細については、「[言葉の定義](/rules/00-word-definition/#センシティブ)」および「[モデレーションポリシー](/rules/01-moderation-policy/)」のページをご確認ください。
</Aside>

# クリエイターの方へのお願い

## R-18指定画像のアップロード禁止について
性器の露出や、その他過度な性的表現、暴力表現などを含む画像は**R-18画像**と判断し、予告なく削除する場合があります。

詳細については、「[言葉の定義](/rules/00-word-definition/#センシティブ)」および「[モデレーションポリシー](/rules/01-moderation-policy/)」のページをご確認ください。

## JASRAC・NexTone管理楽曲のアップロード禁止について
「ぼすきー」は、JASRAC（一般社団法人日本音楽著作権協会）およびNexTone（株式会社NexTone）といった著作権管理団体との間で包括的な利用許諾契約を締結していません。
そのため、これらの団体が管理する楽曲の音源や歌詞などのアップロードは禁止しています。発見した場合、予告なく削除することがあります。

詳細については、「[JASRAC・NexTone管理楽曲について](/rules/05-jasrac-nextone-songs/)」のページをご確認ください。

<Aside type="note" title="推奨する投稿方法">
ニコニコ動画やYouTubeなど、著作権管理団体と包括契約を締結しているサービスにアップロードされた楽曲へのリンクを投稿することは問題ありません。
</Aside>
