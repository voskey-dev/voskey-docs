---
title: 絵文字申請におけるライセンス分類について
---

カスタム絵文字を申請する際には、その絵文字の利用範囲や条件を示すライセンス情報を明記していただく必要があります。以下の「大分類」と「許可分類」から適切なものを選択し、「大分類／許可分類」の形式（全角スラッシュ区切り）で記載してください。

## 大分類：絵文字の作成方法に基づいた分類

| 大分類名       | 対象となるケース                                                                                                                                                              |
| -------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **自作**       | ご自身で一から作成した画像。加工、キャラクター、楽曲などの要素を含まない純粋な自作の場合（インターネット・ミームなどを元ネタとする場合もこちらに該当）。                                                                 |
| **加工**       | ご自身で作成したが、素材として第三者が作成した画像を含んでいる場合。※ただし、素材の権利者が絵文字としての利用を許諾している場合、購入した素材、またはクレジット表記不要のフリー素材の場合は「自作」として扱って構いません。 |
| **キャラ**     | ご自身で作成したが、デザインに既存のキャラクターの要素が含まれている場合。※絵文字の文字部分にキャラクター名が含まれる程度であれば、この分類にする必要はありません。                                                               |
| **歌**         | ご自身で作成したが、合成音声に関連する楽曲（いわゆる「ボーカロイド楽曲」など）の要素が含まれている場合。                                                                                                     |
| **ジェネレータ** | Webサービス（MEGAMOJIなど）を利用して生成した画像。※許可分類を省略した場合、自動的に「元ネタのみ」として扱われます。元サービスがCC-BYなどのライセンス表示を求めている場合は、その旨を明記してください。                               |
| **借用**       | 申請者自身が画像の権利を所有しておらず、かつ権利者の代理として申請しているわけでもない場合。※権利者の代理として申請し、その権利者が自作した画像であれば「自作」として扱って構いません。                                                 |

**複数の要素を含む場合：** 「加工／キャラ／歌」のように、該当する大分類をスラッシュで繋げて記載してください。

## 許可分類：絵文字の利用範囲や条件に関する分類

| 「大分類」の対象 | 許可分類名       | 連合 | 内容                                                                                                                                                                                                                                                           |
| -------------- | ---------------- | ---- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 自作系         | **連合不可**     |  ×   | 「ぼすきー」サーバー内でのみ絵文字として利用を許可し、他サーバーでの表示（連合）や利用は許可しません。                                                                                                                                                                             |
| 自作系         | **連合のみ可**   |  ○   | 「ぼすきー」サーバー内で絵文字として利用し、他サーバーでの表示（連合）も許可します。ただし、他サーバーがこの絵文字を自身のサーバーに登録して利用することは許可しません（あくまで意思表示となります）。                                                                                                 |
| 自作系         | **再利用可**     |  ○   | 「ぼすきー」サーバー内で絵文字として利用し、他サーバーでの表示（連合）も許可します。さらに、一定の条件（例：作者名やアカウントの明記）を満たせば、他サーバーでの絵文字としての利用も許容します。詳細な条件がある場合や、別途配布場所がある場合は、そのURLを記載してください（例：ご自身のウェブサイト、ニコニコ静画など）。動画素材としての利用なども含めたい場合は、この分類を推奨します。 |
| 自作系※        | **元ネタのみ**   |  ○   | 作者としては絵文字画像を自由に使って構わないという意思表示です。ただし、キャラクターや素材を利用しているため、それらの元ネタの権利関係を確認し、規約があればそれを遵守することを求めます。**大分類が「自作」以外の「キャラ」「加工」「ジェネレータ」などの場合に主に用います。**                                                                                             |
| 自作系※        | **パブリック**   |  ○   | 作者としては絵文字画像を自由に使って構わないという意思表示であり、作者名の表示も不要です。ただし、この絵文字に起因するいかなる問題についても作者は責任を負いません。（大分類が純粋な「自作」の場合のみ選択可能です。）**⚠️ 大分類が「キャラ」などの場合、申請が「パブリック」であっても、運営側で自動的に「元ネタのみ」に修正します。**                                                                 |
| 借用           | **再配布不可**   |  ×   | 画像の利用は許諾されているものの、再配布は許可されていません。そのため、他サーバーへの送信（連合）は行いません。                                                                                                                                                                   |
| 借用           | **再配布可**     |  ○   | 画像の再配布が許諾されている、またはパブリックドメイン（著作権保護期間満了など）の画像です。                                                                                                                                                                           |
| 借用           | **絵文字利用可** |  ○   | 画像の配布元が、絵文字としての利用を明確に許可しています。                                                                                                                                                                                                 |
| 借用           | **その他**       |  ？  | 上記のいずれにも当てはまらない画像。YouTubeチャンネルのアイコン、Skebで依頼したイラスト、地方自治体のロゴマークなどを想定しています。この場合は運営が個別に利用可否を判断します。                                                                                                                   |

### ライセンス表記のサンプル

- `自作／再利用可 配布場所 https://seiga.nicovideo.jp/seiga/imXXXXXXXX`
- `自作／再利用可 要：作者の表示`
- `借用／絵文字利用可`
- `ジェネレータ` （この場合、許可分類は自動的に「元ネタのみ」となります）
- `ジェネレータ／元ネタのみ`
- `ジェネレータ／連合不可`
- `自作／パブリック`
- `加工／連合不可 再配布不可の素材を利用しているため`
- `キャラ／元ネタのみ`
- `キャラ／歌／連合のみ可`

# 「連合」に関する重要な注意事項

## 「連合」とは？
「ぼすきー」は、Misskeyというソフトウェアを利用して構築されたSNSです。Misskeyは、ActivityPubという共通の仕組み（プロトコル）を用いることで、他のMisskeyサーバーやMastodonインスタンスなど、異なるサーバー間でユーザー同士がフォローし合ったり、メッセージを送り合ったりすることができます。このように、**異なるサーバー間でデータをやり取りできる状態やつながり**を「連合」と呼びます。

## 絵文字を「連合可」にする場合と「連合不可」にする場合の違い
- **連合可の場合：** 作成した絵文字は、「ぼすきー」サーバー内だけでなく、連合している他のサーバーのユーザーにも表示されます。
- **連合不可の場合：** 作成した絵文字は、「ぼすきー」サーバー内でのみ表示されます。連合先の他のサーバーでは、その絵文字は表示されず、代わりに「いいね」などの汎用的なリアクションに置き換えられます（過去に一度でも連合されたことがある絵文字は、絵文字名がテキストとして表示される場合があります）。
  ※ Mastodonなど、一部のSNSでは絵文字リアクション機能自体に対応していないため、「連合」の設定に関わらず「お気に入り」などに置き換えられます。

## 「連合不可」を選択するケースについて
Misskeyには、**他のサーバーで公開されている絵文字を、自身のサーバーの絵文字として簡単に取り込む機能（インポート機能）** が存在します。しかし、現在のバージョン（v13.14.2時点）では、インポート時にライセンス情報を確認する仕組みが十分に整備されていません。

これにより、**ご自身が作成した絵文字が、意図しない形で他のサーバーで利用されてしまうリスク**が生じます。規模の大きなサーバーや歴史の長いサーバーは、権利関係に比較的慎重な傾向がありますが、小規模なサーバーや新しいサーバーでは、権利意識が低いユーザーによって安易にインポートされてしまう可能性も否定できません。

「連合不可」に設定することで、Misskeyのインポート機能による取り込みを防ぎ、このようなリスクを低減することができます。ただし、その代償として、作成した絵文字は他のサーバーでは表示されなくなります。**残念ながら、現時点では「他のサーバーには表示させたいが、インポートは防ぎたい」という設定はできません。**

**「自作／連合のみ可」という設定は、あくまで申請者の意思表示であり、システム的に意図しない利用を防ぐ強制力を持つものではない**ことをご理解ください。

## 「連合不可」にすれば、無断転載を完全に防げますか？
「連合不可」に設定すれば、Misskeyのインポート機能による安易な取り込みは防げるため、一定の効果は期待できます。しかし、悪意のあるユーザーが画像ファイルを直接保存し、自身のサーバーに手動で登録する形での無断転載は、技術的に完全に防ぐことは困難です。これはMisskeyに限らず、インターネット上で公開されているあらゆる画像に共通する課題です。

## 絵文字が意図せず利用された場合の「ぼすきー」運営の対応
万が一、ご自身が作成した絵文字が他のサーバーで不正に利用されていることを発見した場合、その状況を完全に是正することは現実的に非常に困難です。その上で、「ぼすきー」運営として可能な対応は以下の通りです。

- 不正利用が確認されたサーバーの管理者に対し、対話を通じて是正を要求します。
- 対話による解決が困難な場合は、当該サーバーとの間でトラブルが発生している旨を「ぼすきー」内で周知し、必要に応じてサーバー単位でのブロック措置を検討します。

運営として可能な限りの対応は行いますが、必ずしもご期待に沿える結果をお約束できない場合があることを、あらかじめご了承ください。

## 絵文字申請に伴う許諾事項
絵文字を申請した時点で、その絵文字の「連合」設定の可否に関わらず、**「ぼすきー」運営および「ぼすきー」の全ユーザーに対し、当該画像の公衆送信権（絵文字としての利用権）を許諾したもの**とみなします。

「連合可」として絵文字を申請した場合、上記に加えて、**「ぼすきー」を含む全てのMisskeyサーバーの運営団体およびユーザーに対し、当該画像の公衆送信権（絵文字としての利用権）を許諾したもの**とみなします。

また、一度「連合」を通じて他のサーバーに絵文字情報が渡った後に、ライセンス情報などを変更しても、**他のサーバーにその変更内容を即座に反映させることはできません。** この点を許容できない場合は、「連合不可」を選択することを強く推奨します。

### 最後に（管理者より）
権利関係の話題は複雑で、慎重な対応が求められます。特に二次創作が活発なコミュニティにおいては、様々なケースが想定されます。

一方で、合成音声というジャンルは、多くのクリエイターが二次創作の画像や音声を素材として公開し、それらを活用して新たな作品が生み出されるという、創造的な連鎖によって発展してきた側面も持ち合わせています。

「ぼすきー」としては、クリエイターの権利を尊重しつつも、このような創作文化の火を絶やさぬよう、バランスの取れた運営を心掛けていきたいと考えています。そのためにも、申請者の皆様には、ライセンスに関する明確な意思表示にご協力をお願いするとともに、万が一トラブルが発生した際には、運営としても可能な範囲でサポートさせていただきます。
