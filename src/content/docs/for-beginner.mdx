# 投稿する際に気をつけて欲しいこと

- センシティブな文章は、パブリック（🌐）でそのまま公開しないで下さい。
  - 隔離を行った上で公開して下さい。例えば、「内容を隠す👁️」機能を活用し、センシティブであることを注釈に明記する等の方法があります。
- センシティブな画像や動画は、投稿前に「👁️センシティブとして設定」機能を使用し、センシティブフラグを設定してから投稿して下さい。

<Aside type="note" title="「センシティブ」とは？">
- 「見たくない人がいる可能性のある要素のこと」です。
  - NSFW（Not Safe For Work）と呼ばれることもあります。職場、学校、電車の中など公共の場で閲覧しないほうがい内容という意味です。
  - 一般的に性的な内容と捉えられることが多いですが、暴力・政治・宗教、激しい点滅を伴う内容等も含まれます。
  - 詳細は「[言葉の定義](rules/important/00-word-definition/#センシティブ)」「[モデレーションポリシー](rules/important/01-moderation-policy/)」ページをご覧下さい。

</Aside>

# クリエイターの方に気をつけて欲しいこと

## R-18画像のアップロードは禁止です
局部が写っていたり、その他過激な要素が含まれる画像は**R-18画像**と判断し、予告無く削除させて頂く場合があります。

詳細は「[言葉の定義](rules/important/00-word-definition/#センシティブ)」「[モデレーションポリシー](rules/important/01-moderation-policy/)」ページをご覧下さい。

## JASRAC・NexTone管理楽曲のアップロードは禁止です
ぼすきーは上記の著作権管理団体と契約を結んでいない為、上記音源・歌詞などのアップロードは禁止しています。発見した場合は、予告無く削除させて頂く場合があります

詳細は「[JASRAC・NexTone管理楽曲について](rules/important/05-jasrac-nextone-songs/)」ページをご覧下さい。

<Aside type="note" title="推奨する方法">
ニコニコ動画やYoutube等でアップロードした楽曲をリンクとして投稿するのは構いません。
</Aside>
